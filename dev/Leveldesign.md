# Level design

Level 1: 
- simple solution in  moves
- board size: 10x10

Level 2:
- teaches, that you robots block other robots
- solution in 3 moves
- board size: 10x12

Level 3:
- teaches, that robots are on the path to the target, and that you have to move it away first
- solution in 3 moves
- board size: 12x12
